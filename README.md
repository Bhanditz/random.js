Random number generator of statistical distributions.

# How to Use
## Browsers

    <script type="text/javascript" src="/path/to/random.js"></script>
    
    <!-- html -->

    <script type="text/javascript">
        var newPoint = random.normal(0, 1);
    </script>

## Node

    var random = require('/path/to/random');
    var newPoint = random.normal(0, 1);
